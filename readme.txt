== Description ==

A plugin which helps you to manage your online transactions via Money Lover app.

= Key features =

* Manage online Transactions via Money Lover App
* Intergrated Easy Digital Downloads

= Support =
We provide support both on support forum on Wordpress.org and our [support page](http://www.designwall.com/question/) on DesignWall.

== Installation ==

1. Upload `money-lover` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Log in to your WordPress Dashboard and go to menu `Dashboard > Tools > Money Lover` and login into MoneyLover to create a new wallet.
